# 蕴瑜课堂手动启动循环播放下一集

这是一个 Tampermonkey 用户脚本：需要手动启动，然后在当前视频自然结束后自动进入下一集，并在新页面尝试自动播放。

## 功能概览

- 默认不启动，需要手动点击右下角“启动”。
- 启动后只等待播放结束，不会立刻自动播放当前页。
- 当前视频自然结束后自动进入下一集。
- 切到下一集后会进行有限次数的自动播放重试。
- 切集后播放会执行一次短流程：随机播放约 5s -> 随机暂停约 5s -> 回到 00:00 -> 重新播放。
- 自动处理“按住通过”课堂注意力校验。
- 尝试保持页面前台/可见状态，减少后台播放被限制的情况。
- 不倍速、不跳过视频、不伪造观看进度、不过度常驻扫描。

## 使用步骤

1. 安装 Tampermonkey。
2. 新建脚本。
3. 将 `yunyu-autoplay-next.user.js` 的内容粘贴进去并保存。
4. 打开蕴瑜课堂视频页，点击右下角“启动”。

## 行为说明

- 结束信号来自播放器的“播放结束”事件，避免依赖持续轮询。
- 下一集优先从课程索引或“下一活动”链接中查找。
- 自动播放会有限次重试，失败则提示手动播放。
- 切集后的短流程含随机误差，回退时间固定为 00:00。
- 如果出现“按住通过”弹窗，会自动按住并释放。
- 会尝试重写页面可见性与焦点状态，尽量保持课堂网页在前台。

## 注意事项

- 浏览器对后台页面的播放限制较严格，即使脚本尝试保持可见，仍建议把课堂页面放在前台以保证稳定。

## 文件
