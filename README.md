# 蕴瑜课堂手动启动循环播放下一集

这是一个 Tampermonkey 用户脚本，用于在蕴瑜课堂视频页手动启动后，等待当前视频自然结束，自动切换到下一集，并在切集后的新页面自动播放。

## 功能

- 默认不启动，需要手动点击右下角“启动”。
- 当前页启动后只等待播放结束，不会立刻自动播放当前页。
- 当前视频自然结束后自动进入下一集。
- 自动切到下一集后尝试自动播放，然后继续等待结束。
- 不使用常驻扫描，不倍速，不跳过视频，不伪造观看进度。

## 安装

1. 安装 Tampermonkey。
2. 新建脚本。
3. 将 `yunyu-autoplay-next.user.js` 的内容粘贴进去并保存。
4. 打开蕴瑜课堂视频页，点击右下角“启动”。

## 文件

- `yunyu-autoplay-next.user.js`：用户脚本主体。
